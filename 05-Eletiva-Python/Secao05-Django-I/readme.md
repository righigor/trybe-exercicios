### Descrição

Nas Seções 5 e 6 aprenderei a criar aplicações web com Python utilizando o framework Django. O Django é um framework web mais completo e rígido, que traz diversas funcionalidades prontas para serem utilizadas em aplicações web. Com ele, vou poder criar aplicações web com autenticação de usuários, banco de dados, formulários, e muito mais, com apenas algumas configurações!

Tendo isso em mente aqui você encontrará os exercícios que desenvolvi para fixar os conhecimentos que adquiri ao longo desse bloco. Além disso também desenvolvi o projeto ao final de bloco para avaliar tudo o que havia aprendido, você pode acessá-lo [aqui](https://github.com/righigor/trybe-exercicios/tree/main/05-Eletiva-Python/Secao05-Django-I/Dia06-Projeto-SpotNews).
