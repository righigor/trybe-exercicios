### Descrição

Projeto feito em grupo de 5 pessoas. Com o objetivo é colocar em prática tudo o que aprendi sobre React Context API e React Hooks até aqui enquanto pratico a organização de um projeto com a metodologia agile Kanban. O projeto consiste em desenvolver um sistema que permite visualizar, buscar, filtrar, favoritar, compartilhar e acompanhar o processo de preparação de receitas e drinks. A base de dados serão 2 APIs distintas, uma para comidas e outra para bebidas. O layout tem como foco dispositivos móveis, então todos os protótipos vão estar desenvolvidos em telas menores.
