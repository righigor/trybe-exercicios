### Descrição

Hoje aprenderei sobre formulários em HTML, que são uma forma de enviar informações para outra página do seu site ou para um servidor.

### Aprendizados

- Criar formulários em HTML com as tags:
  - `input`;
  - `button`;
  - `textarea`;
  - `select`;
  - `form`.
