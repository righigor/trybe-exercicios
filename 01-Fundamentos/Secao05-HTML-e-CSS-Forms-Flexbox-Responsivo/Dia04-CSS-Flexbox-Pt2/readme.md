### Descrição

Após aprender a posicionar na tela um container flex, agora vou conseguir posicionar também os itens dentro desse container. As propriedades que ajustam o posicionamento dos itens contidos no container flex são muito importantes para que eu consiga ter controle total sobre o layout criado.

### Aprendizados

- Propriedades que alteram o comportamento dos itens flexíveis, permitindo controlar o layout e o comportamento de cada item dentro do container.
