### Descrição

Hoje, vou aprender a utilizar as media queries, que são amplamente utilizadas no design responsivo para exibir diferentes layouts em diferentes telas, a depender do tamanho e/ou da orientação.

### Aprendizados

- Como criar regras CSS específicas para o tamanho e a resolução do dispositivo;
- Como construir páginas que se adaptam à orientação da tela.
